I18NHelper - MT::I18N block tags and modifiers.

= Overview =

I18NHelper plugin provides MT::I18N block tags and modifiers.

= Installation =

 * Download and extract I18NHelper-<version>.zip file.

 * Upload or copy the contents of "plugins" directory into your
   "plugins" directory.

 * After proper installation, you will find "I18N Helper" plugin
   listed on the "System Plugin Settings" screen.
